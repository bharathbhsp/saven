# Saven — App overview (one page)

**Saven** is a spend-tracking app for a group of people. You organise spending into **groups** (e.g. Household, Travel), record **transactions** with amount, date, and category, and can view totals, filter by period, and export. The **Telegram bot SavenTrackingBot** helps you add spend and see summaries from your phone or any device without opening the app—useful when you’re on the go. Search for **SavenTrackingBot** in Telegram to get started. Access is **invite-only**: your admin creates accounts and shares the sign-in link.

---

## What you can do

**Dashboard** — Pick a group, see this month’s total and the last 5 transactions. Create new groups. Quick links to add a transaction or open the full list.

**Transactions** — Add spend (group, amount, date, category, optional note). View all transactions for a group; filter by day, month, or date range (newest first). See who added each entry and when. Export the current view as **CSV** or **PDF**.

**Groups** — Create groups and manage **members** by email (add or remove). Only people with a Saven account can be added. Everyone in a group can add transactions and see the same data.

**Categories** — View categories per group (e.g. Food, Transport, Other). Add group-specific categories; use them when adding transactions.

**Profile** — View your account, sign out, and manage **Telegram**: link your account to **SavenTrackingBot** with a one-time code; set a default group for the bot; or link a **Telegram group** to a Saven group so everyone in that chat can record into the same bucket (each message is attributed to the sender).

**Telegram bot (SavenTrackingBot)** — Search for **SavenTrackingBot** in Telegram. Helps you track spend without opening the app: add spend by typing amount and category (or free text like “50 coffee”); ask for **today**, **month**, or **range** summaries in chat. Handy on the go. After linking your account in the app, the bot uses the group you mention or your linked/default group.

## In short

| Function | Description |
|----------|-------------|
| Sign in | Email + password (invite required). |
| Groups | Create; add/remove members by email. |
| Transactions | Add (web or Telegram); view by day/month/range; export CSV/PDF; see who submitted. |
| Categories | View; add per group; use when adding transactions. |
| Telegram bot (SavenTrackingBot) | Search for SavenTrackingBot in Telegram; link in Profile. Add spend and get summaries in chat. Set default group or link a Telegram group to a Saven group. |
